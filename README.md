# Housing-Price-Prediction-Analysis

## 
The goal of this project is to analyze housing data and build a machine learning model to predict property prices based on key features like area, location, and house attributes.

## Tools & Technologies
- Python
- Pandas, NumPy
- Scikit-learn 
- Matplotlib, Seaborn

## 
- Data Cleaning & Preprocessing
- Feature Engineering (Encoding categorical variables)
- Model Building using Linear Regression & Random Forest
- Model Evaluation using MAE, RMSE, R² Score

##  Key Insights
- Area, location, and number of rooms significantly impact house prices
- Model achieved near-perfect accuracy (R² ≈ 1), indicating strong feature relationships
- Feature importance analysis identified key drivers of pricing

##  Model Performance
- MAE: ~0
- RMSE: ~0
- R² Score: ~1.0

## Visualization
Included a scatter plot comparing actual vs predicted prices to evaluate model performance.

##  Business Impact
- Helps real estate platforms estimate accurate property prices
- Supports investment decisions and pricing strategies
- Enables data-driven decision-making in the housing market
